>> [**[ DOWNLOAD THE APK FROM THE LATEST RELEASE ]**](https://github.com/packetinspector/jellyfin-androidtv-upscaled/releases/latest) <<

If you already have the official client installed, you might get a signature
mismatch. That's fine!! Just uninstall the old debug build first:

```bash
adb uninstall org.jellyfin.androidtv.upscaled
```

Then install away.

### `;========== METHOD 2: BUILD IT YOURSELF (THE COOL KID WAY) ==========`

```bash
;----------------------------------------------------------------
; REQUIRED: Docker. That's it. We don't need no SDKs.
;----------------------------------------------------------------

git clone https://github.com/packetinspector/jellyfin-androidtv-upscaled.git
cd jellyfin-androidtv-upscaled
git checkout custom-fork

./build.sh build           ;  build a debug APK (R8 enabled)
./build.sh release         ;  build a release APK
./build.sh test            ;  run the unit tests
./build.sh clean           ;  scrub the build cache
./build.sh shell           ;  drop into a build container shell
```

Output APK lands at:

```
app/build/outputs/apk/debug/jellyfin-androidtv-v0.0.0-dev.1-debug.apk
```
